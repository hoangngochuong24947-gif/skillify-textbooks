# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-03-15

### Added
- Initial release of skillify-textbooks
- Core skill definition: `book-to-skills`
- Five-stage query pack system for book extraction
- Subagent playbooks for 5 specialized roles
- MethodCard schema for standardized method documentation
- Output schemas for 6 Markdown artifacts
- Parallel execution manifest for optimized agent orchestration
- Agent orchestrator specification for parallel/serial mode switching
- Baseline test scenarios for quality validation
- Visual infographic specification
