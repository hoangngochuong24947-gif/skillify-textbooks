## 一、投资决策路由

### 场景 1.1: 单一独立项目评估

**触发条件**
- 关键词："这个项目值得投资吗？"、"评估一个项目"、"NPV计算"
- 输入：项目现金流预测、初始投资

**路由决策**
```
IF 现金流预测可靠 AND 折现率确定:
    → 使用 MethodCard 1: NPV分析法
    → 同时计算 MethodCard 2: IRR 作为参考
ELSE IF 折现率不确定:
    → 先执行 MethodCard 7: CAPM估算资本成本
    → 再执行 MethodCard 1: NPV分析
```

**输出**
- NPV值及决策建议
- IRR及与要求回报率比较
- 敏感性分析（可选）

---

### 场景 1.2: 互斥项目选择

**触发条件**
- 关键词："两个项目选哪个"、"互斥项目"、"规模不同"
- 输入：多个项目现金流

**路由决策**
```
IF 项目规模相同 AND 期限相同:
    → NPV最高者优先
ELSE IF 项目规模不同:
    → 计算盈利指数 (PI = NPV/初始投资)
    → 或增量IRR分析
ELSE IF 期限不同:
    → 使用约当年均成本法 (EAC)
    → 或最小公倍数法
```

**冲突解决**
- NPV vs IRR冲突：NPV优先
- IRR vs MIRR冲突：MIRR更可靠

---

### 场景 1.3: 高风险/不确定性项目

**触发条件**
- 关键词："市场不确定"、"风险很高"、"最坏情况"
- 输入：项目现金流范围、关键变量

**路由决策**
```
IF 需要了解价值范围:
    → MethodCard 4: 情景分析法
    → 输出：乐观/基本/悲观情景NPV

IF 需要识别关键变量:
    → MethodCard 5: 敏感性分析法
    → 输出：龙卷风图、关键变量排序

IF 需要最低可行销量:
    → MethodCard 6: 盈亏平衡分析
    → 输出：财务盈亏平衡点

IF 以上全部需要:
    → 顺序执行：情景分析 → 敏感性分析 → 盈亏平衡
```

---

### 场景 1.4: 战略投资/研发项目

**触发条件**
- 关键词："研发投资"、"分阶段投资"、"可以放弃"、"扩张机会"
- 输入：项目现金流 + 未来选择权

**路由决策**
```
IF 存在管理灵活性:
    → MethodCard 15: 实物期权估值法

    识别具体期权:
        - 扩张期权 → 看涨期权估值
        - 放弃期权 → 看跌期权估值
        - 延迟期权 → 美式期权估值
```

**与传统NPV结合**
```
战略NPV = 传统NPV + 实物期权价值
```

## 二、估值问题路由

### 场景 2.1: 企业整体估值

**触发条件**
- 关键词："公司值多少钱"、"收购估值"、"企业价值"
- 输入：财务报表、行业信息

**路由决策**
```
主流程:
    → MethodCard 3: DCF估值模型

    子任务分解:
        1. MethodCard 9/10: 财务分析（历史业绩）
        2. Forecast-Generator: 未来现金流预测
        3. MethodCard 7: CAPM估算权益成本
        4. DCF计算: 预测期现值 + 终值

辅助验证:
    → 相对估值法: EV/EBITDA, P/E, P/B
```

---

### 场景 2.2: 上市公司股票估值

**触发条件**
- 关键词："股票价值"、"是否被高估"、"目标价"
- 输入：股价、财务报表、市场数据

**路由决策**
```
方法一: DCF
    → MethodCard 3: DCF模型
    → 使用股利折现模型(DDM) 或 自由现金流(FCFE)

方法二: 相对估值
    → 财务比率: P/E, EV/EBITDA
    → 与行业平均比较

综合判断:
    IF DCF价值 > 市场价格 AND 相对估值合理:
        → 低估，建议买入
    ELSE IF DCF价值 < 市场价格:
        → 高估，建议卖出
```

---

### 场景 2.3: 债券估值

**触发条件**
- 关键词："债券价格"、"到期收益率"、"YTM"
- 输入：票面利率、面值、到期日、市场利率

**路由决策**
```
→ 标准DCF方法:
    债券价值 = ∑[利息/(1+r)^t] + 面值/(1+r)^n

特殊情况:
    IF 零息债券:
        → 按面值折现
    IF 可转换债券:
        → 债务价值 + MethodCard 15: 期权价值
    IF 含权债券:
        → 基础价值 ± 期权价值调整
```

## 三、风险分析路由

### 场景 3.1: 投资项目风险评估

**触发条件**
- 关键词："项目风险多大"、"最坏会怎样"、"风险因素"
- 输入：项目现金流预测、变量假设

**路由决策**
```
定量分析:
    → MethodCard 4: 情景分析 (3种情景)
    → MethodCard 5: 敏感性分析 (单一变量)
    → MethodCard 6: 盈亏平衡分析 (最低要求)

定性分析:
    → 识别风险类型:
        - 市场风险 → 不可分散，需要风险溢价
        - 执行风险 → 管理质量评估
        - 监管风险 → 政策变化影响

输出:
    - NPV分布范围
    - 风险调整后NPV
    - 风险缓解建议
```

---

### 场景 3.2: 投资组合构建

**触发条件**
- 关键词："401k投资"、"资产配置"、"组合优化"
- 输入：可选资产、投资者风险偏好

**路由决策**
```
Step 1: 资产分析
    → MethodCard 7: CAPM计算预期收益
    → MethodCard 8: 多元化分析各资产贡献

Step 2: 组合构建
    → 有效前沿计算
    → 最优风险-收益组合

Step 3: Beta管理
    → 组合Beta计算
    → 根据市场观点调整系统性风险暴露
```

---

### 场景 3.3: 资本成本计算

**触发条件**
- 关键词："WACC"、"资本成本"、"折现率"、"要求回报率"
- 输入：资本结构、债务成本、税率

**路由决策**
```
债务成本:
    → 现有债务: 使用YTM
    → 新债务: 参考市场利率或信用评级

权益成本:
    → MethodCard 7: CAPM模型
        - Rf: 10年期国债收益率
        - Beta: 回归估计或行业平均
        - 市场风险溢价: 5-7%（历史或前瞻）

    替代方法:
        → 股利增长模型: r = D1/P0 + g

WACC计算:
    WACC = (E/V)×Re + (D/V)×Rd×(1-Tc)
```

## 四、财务分析路由

### 场景 4.1: ROE变化分析

**触发条件**
- 关键词："ROE为什么下降"、"盈利能力分析"、"ROE分解"
- 输入：多年财务报表

**路由决策**
```
→ MethodCard 9: 杜邦分析法

三因素分解:
    ROE = 净利润率 × 总资产周转率 × 权益乘数

五因素扩展（如有需要）:
    ROE = 税收负担 × 利息负担 × EBIT利润率 × 周转率 × 杠杆

分析步骤:
    1. 计算本期 vs 上期各因素
    2. 识别变化最大的驱动因素
    3. 深入分析该因素背后的业务原因
    4. 与行业平均对比
```

---

### 场景 4.2: 公司财务健康诊断

**触发条件**
- 关键词："财务健康吗"、"会不会破产"、"偿债能力"
- 输入：资产负债表、利润表、现金流量表

**路由决策**
```
全面分析:
    → MethodCard 10: 财务比率分析

五大维度:
    1. 流动性: 流动比率、速动比率
    2. 杠杆: 负债率、利息保障倍数
    3. 营运: 周转率、现金周期(MC 13)
    4. 盈利: ROA、ROE、毛利率
    5. 市场: P/E、EV/EBITDA

评分系统:
    - 与行业平均比较
    - 趋势分析（3-5年）
    - 识别红旗指标

深度分析（如需要）:
    → MethodCard 9: 杜邦分析深入ROE
    → MethodCard 13: 现金周期分析营运效率
```

## 五、资本结构路由

### 场景 5.1: 最优资本结构确定

**触发条件**
- 关键词："应该借多少钱"、"债务比例"、"最优资本结构"
- 输入：当前资本结构、EBIT预测、行业数据

**路由决策**
```
主要方法:
    → MethodCard 11: 权衡理论模型

    计算:
        - 税盾价值 = Tc × D
        - 困境成本 = 概率(违约) × 违约损失
        - 最优D/E = 边际税盾收益 = 边际困境成本

辅助分析:
    → MethodCard 12: MM定理（理解理论基础）

实际考虑:
    - 行业基准
    - 信用评级目标
    - 财务灵活性需求
    - 信号效应（信息不对称）
```

---

### 场景 5.2: 融资方式选择

**触发条件**
- 关键词："发行股票还是债券"、"融资决策"、"怎么筹资"
- 输入：融资需求金额、当前资本结构

**路由决策**
```
分析框架:
    1. 啄序理论检查:
        - 内部融资是否充足？
        - 债务容量是否还有空间？
        - 股权融资作为最后选择

    2. 成本比较:
        - 债务成本 vs 权益成本
        - 考虑税收优势

    3. 控制权考虑:
        - 股权稀释影响
        - 债务契约限制

    4. 信号效应:
        - 发行股票可能被视为负面信号
        - 债务发行显示管理层信心

决策树:
    IF 有内部资金:
        → 优先使用
    ELSE IF D/E < 目标水平:
        → 考虑债务融资
    ELSE IF 需要保持灵活性:
        → 考虑可转债或混合工具
    ELSE:
        → 股权融资
```

## 六、营运资本路由

### 场景 6.1: 营运资本优化

**触发条件**
- 关键词："改善现金流"、"减少营运资本"、"提高效率"
- 输入：营运资本各组成部分数据

**路由决策**
```
诊断:
    → MethodCard 13: 现金周期管理法

    计算:
        - 经营周期 = 存货天数 + 应收天数
        - 现金周期 = 经营周期 - 应付天数

优化策略:
    1. 存货管理:
        - IF 存货周转慢:
            → 实施JIT系统
            → 优化EOQ（MethodCard 14相关）

    2. 应收账款:
        - IF DSO过高:
            → 收紧信用政策(MC 14)
            → 提供早付折扣
            → 考虑保理

    3. 应付账款:
        - IF 付款过快:
            → 在不损害供应商关系前提下延长账期

量化影响:
    - 每减少1天现金周期 = 释放现金 = (销售成本/365)
```

---

### 场景 6.2: 信用政策设计

**触发条件**
- 关键词："信用政策"、"应收账款"、"付款条件"、"坏账"
- 输入：当前信用政策、销售数据、资金成本

**路由决策**
```
→ MethodCard 14: 信用政策分析法

决策变量:
    1. 信用标准: 多严格的客户筛选
        → 信用评分系统

    2. 信用期间: 30天？60天？90天？
        → 竞争环境分析

    3. 现金折扣: 2/10 net 30？
        → 成本收益分析

    4. 收款政策: 逾期处理
        → 成本效益权衡

分析步骤:
    1. 估算政策变化的增量销售
    2. 计算增量利润（边际贡献）
    3. 计算增量成本（资金成本+坏账+管理成本）
    4. NPV = 增量收益 - 增量成本
    5. NPV > 0 → 接受新政策
```

### 复杂问题的多方法组合

| **高风险项目** | NPV | 敏感性+盈亏平衡 | 实物期权 |
